De toegankelijke videoplayer is Deze module is ontwikkeld door alterNEt Internet.
Voor een uitleg over het gebruik kunt u terecht in de bron van: index.html

----

Wanneer u deze video-player lokaal wilt testen zal de ondertiteling niet in alle browsers werken
vanuit een security oogpunt: "Cross origin requests are only supported for HTTP."
Lees hier meer over op: https://code.google.com/p/html5security/wiki/CrossOriginRequestSecurity

Dit probleem heeft u niet wanneer u de videoplayer op uw website plaatst.
